Каждая из других допустимых букв может встречаться в слове любое количество раз или не встречаться совсем. Слово не должно начинаться с буквы Ш и оканчиваться гласными буквами. Словом считается любая допустимая последовательность букв, не обязательно осмысленная. Сколько существует таких слов, которые может написать Вася?

4. Генерация всех 6-буквенных комбинаций букв слова 'ВИШНЯ'
5. Проверка условий для каждой комбинации:
   - Буква 'В' встречается не более 1 раза
   - Первая буква не 'Ш'
   - Последняя буква не гласная ('И' или 'Я')
6. Увеличение счетчика при выполнении всех условий

# ЗАДАНИЕ 2
#### Задание: Сколько единиц содержится в двоичной записи значения выражения $`4^{2014} + 2^{2015} - 8`$?

Вычисляется значение арифметического выражения. Результат сохраняется в переменной x. Организуется цикл, который переводит число x в двоичную систему счисления. Так как цифры получались в обратном порядке (последняя цифра стала первой), строка num переворачивается. Методом count("1") подсчитывается количество символов "1" в полученной двоичной строке. В результате на экран выводится количество единиц в двоичной записи исходного числа.

5. Если x попадает в отрезок [400 000 000; 600 000 000], то число добавляется в список.
